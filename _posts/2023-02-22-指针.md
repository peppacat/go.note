# 基本类型
1. 基本类型,变量存的就是值,称为值类型
2. 获取变量的地址,使用&
3. 指针类型,变量存的是一个地址,这个地址指向的空间才是值. var ptr *int = &num
4. 获取指针类型指向的值,使用*. var ptr *int = &num

```go
package main

import "fmt"

func main() {
	var i int = 10
	fmt.Println("i的地址=", &i)
	var ptr *int = &i
	fmt.Printf("i的地址=%v\n", ptr)
	fmt.Printf("ptr的地址=%v\n", &ptr)

	*ptr = 100
	fmt.Printf("i的值=%v\n", i)
}
```
```
i的地址= 0xc000120058
i的地址=0xc000120058
ptr的地址=0xc000144020
i的值=100
```
